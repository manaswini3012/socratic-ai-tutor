# 🧠 Socratic AI Tutor

Socratic AI Tutor is a simple educational chatbot built using Flask and Ollama.

Unlike normal AI systems, this tutor does NOT give direct answers.
Instead, it guides students using questions, hints, and step-by-step thinking to help them learn actively.

---

## 🚀 Features

- Socratic questioning style
- Encourages critical thinking
- Step-by-step guidance
- Friendly tutor-like responses
- Runs locally (No API cost)

---

## 🛠 Technologies Used

- Python
- Flask
- Ollama (LLaMA 3 model)
- HTML
- CSS
- JavaScript

---

## 📦 How To Run This Project

### 1. Install Ollama
Download from:
https://ollama.com

Then run:
ollama run llama3


---

### 2. Install Required Python Packages
python -m pip install -r requirements.txt


---

### 3. Start the Backend Server
python socratic.py

Server runs at:http://127.0.0.1:5000


---

### 4. Open Frontend
Open `index.html` in your browser.
